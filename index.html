Chapter 0: The Portable Lab

A "Live USB" allows you to carry your hacking OS in your pocket. By enabling Persistence, data is saved after reboot.

I. Setup Requirements

  • 8GB+ USB Drive
  • Kali Linux "Live Boot" ISO
  • Rufus (Windows) or Etcher (Mac/Linux)

II. Configuration (Rufus)

  1. Select your USB Device.
  2. Click SELECT and choose Kali ISO.
  3. CRITICAL: Use the "Persistent Partition Size" slider. Allocate max space (leave 1GB empty).
  4. Click START (ISO Image Mode).

III. Booting Instructions

Tap the Boot Menu Key immediately after power on:

Dell: F12
HP: F9
Lenovo: Enter/F12
Asus: Esc
Warning: On the Kali Splash Screen, do not hit Enter. Use arrows to select Live System with USB Persistence.

Chapter 2: OpSec & Anonymity

Before you scan, you must cloak. We use MAC Spoofing (Local) and ProxyChains (Network).

Level 1: MAC Spoofing

# 1. Take network down
$sudo ifconfig wlan0 down

# 2. Randomize Hardware ID
$sudo macchanger -r wlan0

# 3. Bring network up
$sudo ifconfig wlan0 up

Level 2: The "Ghost" Ritual (ProxyChains)

Run this before every scan to route traffic through Tor.

# 1. Start the Tor Service
$sudo service tor start

# 2. Check your public IP (Should be different from home)
$proxychains curl ifconfig.me

Module 1: SpiderFoot

AUTOMATED INFRASTRUCTURE MAPPING

Description

An automated OSINT framework that queries 100+ public data sources (WhoIs, DNS, Blacklists) to map a target's infrastructure.

Use Case

  • Asset Discovery (Subdomains/IPs)
  • Threat Intelligence (Blacklists)
  • Data Leaks (Exposed PII)

Installation

git clone https://github.com/smicallef/spiderfoot.git
cd spiderfoot
pip3 install -r requirements.txt

Execution & Data Retrieval

# Launch Web GUI (Recommended)
python3 sf.py -l 127.0.0.1:5001
Visual Workflow: Open browser to http://127.0.0.1:5001.
Output: Results are visualized in graphs. Export as CSV/JSON from the "Export" tab in the dashboard.

Module 2: Photon

WEB CRAWLER & EXTRACTION

High-speed crawler that extracts secrets, keys, and files rather than just indexing pages.

git clone https://github.com/s0md3v/Photon.git

Commands & Data Retrieval
